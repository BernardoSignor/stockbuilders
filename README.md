*Tema/proposta do sistema*

O projeto consiste no desenvolvimento de um sistema web para gerenciamento
de estoque e vendas de uma distribuidora de bebidas. O sistema permitirá controlar
produtos, categorias, clientes, fornecedores, movimentações de estoque e pedidos de
venda através de uma interface administrativa e também de uma área pública para
consulta de produtos.


*Sistema/app/site de inspiração*

Inspiração do tema minha mesmo.
[Tiny ERP](https://tiny.com.br?utm_source) - (Pedi para a IA me passar essa ideia).


*Público-alvo*

Distribuidoras de bebidas, mercados, depósitos e pequenas empresas do ramo
alimentício.


*Lista inicial de requisitos funcionais* 
- Cadastro de usuários
- Login e autenticação
- Cadastro de produtos
- Cadastro de categorias
- Cadastro de fornecedores
- Cadastro de clientes
- Controle de estoque
- Registro de entradas e saídas de produtos
- Cadastro de pedidos
- Dashboard com gráficos
- Relatórios de vendas
- Filtros de pesquisa
- API JSON para consulta de produtos e pedidos


*Justificativa da ideia*

A ideia do sistema surgiu devido à necessidade de organização e controle de
estoque em empresas distribuidoras de bebidas. Muitas pequenas empresas ainda
utilizam controles manuais ou planilhas, dificultando a gestão de produtos e vendas. O
sistema busca facilitar o gerenciamento das informações, melhorar a organização e
fornecer relatórios úteis para tomada de decisão.
